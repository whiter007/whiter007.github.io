三、Forms 表单

  1. fieldset：Set of form controls grouped by theme.
  2. meter：Control for entering a numeric value in a known range.
  3. legend：Define a name for a fieldset.
  4. label：Caption for a form control.
  5. input：Generic form input.
  6. textarea：Multiline free-form text input.
  7. form：Form.
  8. select：Control for selecting from multiple options.
  9. optgroup：Group of option.
  10. option：Single option within a select control.
  11. output：Contains the results of a calculation.
  12. button：A button.
  13. datalist：Define sets of options.
  14. keygen：Generates private-public key pairs.
  15. progress：ontrol for displaying progress of a task.

四、Tabular data 表格数据

  1. col：Columns in a table.
  2. colgroup：Defines a group of columns in a table.
  3. caption：Title of a table.
  4. table：Table of multi-dimensional data.
  5. tr：A row of cells.
  6. td：Table cell.
  7. th：Table heading.
  8. tbody：Contains rows that hold the table's data.
  9. thead：Contains rows with table headings.
  10. tfoot：Contains rows with summary of data.

五、Metadata and scripting 元数据脚本

  1. head：First element of the HTML document. Contains document metadata.
  2. title：Document title.
  3. meta：Document metadata that can't be represented with other elements.
  4. base：Specifies URL which non-absolute URLs are relative to.
  5. link：Other resources related to the document.
  6. style：Styling defined inline data.
  7. noscript：Contains elements that are part of the document only if scripting is disabled.
  8. script：Inline or linked client side scripts.

六、Grouping content 分组内容

  1. br：Line break.
  2. hr：Thematic break within a paragraph.
  3. figcaption：Caption for a figure.
  4. figure：Contains elements related to single concept, such as an illustration or code example.
  5. p：Paragraph content.
  6. ol：Ordered list.
  7. ul：Unordered list.
  8. li：List item.
  9. div：Container with no semantic meaning.
  10. pre：Text that is preformatted in the HTML code.
  11. blockquote：Quote from another source.
  12. dl：List of term-description pairs.
  13. dt：Term which will be described.
  14. dd：Description for the preceding term.

七、Document sections 文档 块

  1. body：Document content.
  2. aside：Content related to surrounding elements that doesn't belong inline, such as a advertising or quotes.
  3. address：Contact information for the current article.
  4. h1-6：Heading for the current section.
  5. section：Contains elements grouped by theme, for example a chapter or tab box.
  6. header：Navigation or introductory elements for the current section.
  7. nav：Contains a collection of links.
  8. article：Section of the page content, such as a blog or forum post.
  9. footer：Footer of the current section.
  10. hgroup：Collection of headings for the current section. The highest ranked heading represents the group in the document outline.

八、Interactive elements 交互元素

  1. menu：Set of commands.
  2. command：Command the user can perform, such as publishing an article.
  3. summary：Caption of a details element.
  4. details：Contains additional information, such as the contents of an accordian view.

九、Embedding content 嵌入内容

  1. img>：An image.
  2. area：Hyperlink area in an image map.
  3. map：Image map for adding hyperlinks to parts of an image.
  4. embed：Reference to non-HTML content.
  5. object：External resource such as an image, iframe or plugin.
  6. param：Parameters for the parent object.
  7. source：Alternative sources for parent video or audio elements.
  8. iframe：Nested browser frame.
  9. canvas：Bitmap which is editable by client side scripts.
  10. track：Specifies external timing track for media elements.
  11. audio：Audio file.
  12. video：Video.
